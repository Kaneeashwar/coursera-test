ABOUT DIV TAG

Introduction:

The div tag is known as Division tag. The div tag is used in HTML to make divisions of content in the web page like (text, images, header, footer, navigation bar, etc). ... It is used to the group of various tags of HTML so that sections can be created and style can be applied to them.

ABOUT SPAN TAG

Introduction:

The <span> HTML element is a generic inline container for phrasing content, which does not inherently represent anything. It can be used to group elements for styling purposes (using the class or id attributes), or because they share attribute values, such as lang.
